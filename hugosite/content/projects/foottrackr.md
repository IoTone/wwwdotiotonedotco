FootTrackr offered a simple way for event organizers and retailers to track movement of customers and count unique visits, in an anonymous and privacy-sensitive way. Using low-cost RFID, FootTrackr was a portable system that could be deployed in small-to-large spaces. Customers needed only a sticker tag — zero enrollment. All data could be monitored in real time, with optional cloud backup.

**Status: EOL.** Code base is available on [GitHub (ProjectGyr)](https://github.com/IoTone/ProjectGyr) for customization.
