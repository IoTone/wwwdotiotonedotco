IoToneKit was an all-in-one starter kit for a low-cost MPU-based solution running Linux on ARMv5 / ARMv7. With an MFi chip integrated, OEMs and systems integrators could quickly start developing for the HomeKit ecosystem with built-in Zigbee support. The process involved defining metadata for each device, reducing development time from months to weeks.

**Status: EOL.** Spiritual successor: [Project MAG\*Net](/projects/project-magnet/), which carries the same protocol-spanning thesis without the HomeKit-specific dependencies.
